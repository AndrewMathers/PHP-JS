Welcome to
Zend OnTheGo

A sample mobile application that monitors your server and gives you live statistics on your server performance.

This mobile app is intented for demo purposes. The application and source code are available with Zend Studio and on GitHub. A contest is open to developers who will add features. Learn more about the contest here.


Monitor

Check the status of your deployed app any time anywhere. See all the issues, from slow requests, function errors, exception, warnings, database errors, etc. You can even share the issues with someone on your contact list.


Stats

See how your server is performing. Get information on memory availability, CPU usage, and more.
