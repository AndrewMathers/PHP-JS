Zend On The Go is a mobile application designed by Zend and written by Julien Guittard, PHP Architect and Zend Framework community member.


The application is a sample application written using Zend Studio. It leverages Zend Framework 2, and the API Builder Apigility. For the backend it uses Zend Server on Amazon Web Services.


The application is designed to demonstrate how to use all the Zend products together to create a mobile application on a robust backend. The application code is available on GitHub (Mobile and Server). A contest is opened to add functionality.
